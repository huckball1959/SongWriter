Songwriter Pro — User Manual

Overview

Songwriter Pro is a chord-and-lyric sheet editor. Build sections and measures on the page, place chords on the chord lane, type lyrics under each row, and use playback to hear chords (with optional strumming and metronome).

Files and tabs

  • File ▾: New Song Sheet, Open (JSON), Save (overwrite linked file when the browser supports it), Save As…, Print, Recent Sheets, and User Manual. After Ctrl+F5, the same tab restores your sheet from session memory even if Keep on Refresh is off.
  • Tabs at the top switch between open songs. Use the ✕ on a tab to close it.

Preferences and startup

  • Preferences ▾: Choose where save dialogs start (Documents, Desktop, a preferred folder you pick, etc.).
  • Keep on Refresh: When ON, your work is saved for the next visit (localStorage). When OFF, new tabs start blank, but Ctrl+F5 in the same tab still restores the current sheet until you close that tab (sessionStorage).
  • D beside toolbar toggles: click D to save that control’s current ON/OFF state as your default for next time.

Sheet header (top of the page)

  • Click the song title area to rename.
  • Click BPM, time signature, or Key in the labeled zones to edit.

Inserting music and rows

  • Insert ▾: Intro… / Verse… etc. add a new block at the end of the sheet. To lengthen an existing Intro (e.g. 1–8 by eight more bars), use Extend section with empty bars… and pick Intro, or Empty measures after… and enter bar 8 (the section that contains that bar grows). Empty measures after… also works from any bar inside the section. Section after measure… cuts and inserts a new preset block; 7. Custom adds a named pill + palette. Instrumental… adds a purple-flag block. Or lyrics-only row.
  • Right-click a measure → Insert measures: empty bars after this bar, Extend this section (at its end), new Verse/Chorus/…, or custom section after this bar.
  • + Page adds about one letter page of empty staff rows so the sheet grows; scroll to see the gray page border.
  • Measures/Line toggles between 2 and 4 measures per row.

Chords

  • Chord lane: Clicking the staff or chord row selects that bar only. To place chords, press Alt+C (with a bar selected) to open Song chords, then drag a variation from the grid or a chip from Used in this song onto the chord lane.
  • Song chords (right dock): The A–G row filters the variation grid. Double-click a variation to add it to the Used list. Right-click a Used chip to remove it.
  • Drag from Song chords to the sheet places the chord on the chord lane. Drops are ignored for lyrics fields so text isn’t polluted.
  • Selected chord: Click a chord to select; click the same chord again to deselect. Alt+←/→ nudges the selected chord along the snap grid when snap is on.
  • Chord dropdown: Default chord for the lane picker and for dragging from Song chords chips.

Snap

  • Snap toggles snapping chord positions to a grid inside each measure.
  • Snap resolution (1/4, 1/8, 1/16) sets chord snap fineness only (invisible subdivisions). Dotted lines follow the time signature.

Playback

  • Space starts or stops playback. Play/Stop does the same.
  • The playhead moves across the chord row. Strum repeats the active chord on each beat (and on offbeats if Offbeat is on). Chords carry across measures and rows until you place a new chord.
  • Metronome clicks beats (reduced when strumming so it doesn’t pile up). Sustain uses a held organ-style sound for preview and playback.
  • Rewind (toolbar) or Alt+R always jumps to measure 1 and selects it. Plain R is left for typing lyrics.
  • Stop: Keep position / Stop: To play start: after you stop, either leave the playhead where you stopped or jump back to where that play began. With Keep position, the next Space plays from the current playhead (your Alt+measure highlight does not pull the playhead back each time). With To play start, each Play can snap to the selected measure and Stop can return you to that play’s start.

Measure selection and where playback starts

  • Contiguous range (most reliable): Right-click inside any measure box (or on a barline). Under Measure range: tap Mark START here on the first bar, then right-click the ending measure (the chord/staff strip or the vertical bar at its right edge) and tap Select measures N–M — the second line updates to show your span. The context menu opens at the pointer and stays on-screen; it scrolls if your window is short.
  • Barlines: Click a vertical bar between measures to select that measure (narrow target). Ctrl+click barlines toggles measures in a multi-select.
  • Click a measure on the canvas or in the lyrics row (when not in Alt+L lyrics mode): selects that bar and moves the playhead. Alt+click a second bar to select the inclusive range from the bar you had selected to that bar. Optional: Shift+Alt+click or Ctrl+click the end measure also completes a range.
  • Selection look: An automatic highlight (e.g. after rewind) uses a light tint and a dashed gray outline. A deliberate selection (click, barlines, section block, range menu) adds a solid red border so it is easy to tell apart. For a multi-measure explicit range, every bar in the range uses that style (including measure 1), not the green playhead ring on the first bar.
  • Alt+←/→ moves the highlighted start by one measure; add Shift to jump by four.
  • Esc clears selection and any range marker. Space plays from the first selected measure when a block is selected.

Sections and lyrics-only rows

  • Choose a Section type and Apply Section to tie measures to a section; section tints and flags appear on the sheet.
  • Click a section flag to select that whole block of measures. Ctrl+click barlines builds multi-measure selections.
  • Right-click any measure → Whole section: pick Select all: Intro (…), Verse 1 (…), etc. — one button per section block (same ranges as the flags).
  • Lyrics-only rows: click the row’s flag to select the block; use context menus or Insert for options.

Context menus

  • Right-click a measure: Range selection, insert measures, delete selected measures (contiguous bars, whole rows), copy, paste, clear, lyrics row, row type, etc.
  • Right-click a chord: Change or delete.
  • Right-click a lyrics-only flag: Edit flag or convert row type.

Keyboard shortcuts (reference)

  • Alt+C — open Song chords (select a bar first). Alt+L — focus lyrics for the selection; plain clicks on lyric fields only select bars until you use this.
  • Ctrl/Cmd+Z undo · Ctrl/Cmd+Y or Shift+Z redo
  • Esc clears selection and closes menus
  • Delete / Backspace: removes a selected chord, a lyrics-only row, or a contiguous measure range (whole staff rows — if your range is partial, you’ll be asked to confirm expanding to full rows). Or use the measure menu → Delete selected measures.
  • Ctrl+C / Ctrl+V with measures selected for copy/paste workflows

Tip: Use the 📖 Manual button on the toolbar anytime. Close this window with Close, the dimmed background, or Esc.
